# Tailwind CSS Expert Skill

This skill focuses on creating a visually "WOW" experience for "Mi Pana" using advanced CSS techniques.

## Core Responsibilities

1.  **Neumorphism Implementation**:
    *   Use dual shadows (light and dark) to create the "extruded" or "inset" look.
    *   Maintain consistent light source (usually top-left).
    *   Example: `shadow-[6px_6px_12px_#b8b9be,-6px_-6px_12px_#ffffff]`

2.  **Claymorphism Implementation**:
    *   Combine rounded corners, multiple inner shadows, and subtle backgrounds.
    *   Achieve the "3D soft plastic" or "clay" effect.

3.  **Color Palette Consistency**:
    *   Maintain the primary and secondary colors extracted from project assets (.psd, .eps).
    *   Ensure accessibility while maintaining soft aesthetic.

4.  **Micro-animations**:
    *   Implement smooth hover and active states for interactive elements.

## Design Tokens

*   Primary Soft: `#f0f0f3` (Typical Neumorphic base)
*   Shadow Dark: `#d1d9e6`
*   Shadow Light: `#ffffff`
