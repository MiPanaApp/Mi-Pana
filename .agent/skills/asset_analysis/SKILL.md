# Asset Analysis Skill

This skill allows for the extraction of design tokens from professional design files.

## Core Capabilities

1.  **Color Extraction**:
    *   Identify primary brand colors from logos and onboarding designs.
    *   Map these to Tailwind CSS color tokens.

2.  **Typography & Styles**:
    *   Identify font styles and effects (like the "graffiti style text effect").
    *   Translate these into CSS/Tailwind utilities.

3.  **Relevant Assets**:
    *   `LOGO/Mi pana.psd`: Primary logo source.
    *   `LOGO/onboarding Mi pana.psd`: Source for onboarding aesthetics.
    *   `39096045_funny_graffiti_style_text_effect.psd`: Reference for text effects.

## Workflow

*   When designing new components, refer to these assets to maintain brand consistency.
*   Use standard image analysis (whenever visible via browser/tools) to match visual styles accurately.
