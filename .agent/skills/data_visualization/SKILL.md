# Data Visualization Skill

This skill provides guidance on presenting marketplace metrics in a clear, interactive, and visually appealing way for administrators.

## Core Responsibilities

1.  **Metric Integration**:
    *   Pull data from Supabase (Sales, New Users, Conversion Rates).
    *   Process and aggregate data for time-series visualization.

2.  **Chart Implementation**:
    *   Use modern libraries (e.g., Chart.js, Recharts) to create responsive graphs.
    *   Ensure charts match the "Mi Pana" aesthetic (soft shadows, consistent colors).

3.  **Interactive Dashboards**:
    *   Implement filtering (by date, category, region).
    *   Ensure real-time updates as new transactions occur.

4.  **Accessibility**:
    *   Maintain high contrast and provide clear labels and tooltips.

## Preferred Chart Types

*   Line Charts: For sales trends over time.
*   Bar Charts: For comparing product categories or seller performance.
*   Doughnut Charts: For user demographic breakdowns.
