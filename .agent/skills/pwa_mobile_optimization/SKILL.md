# PWA & Mobile Optimization Skill

This skill ensures that "Mi Pana" provides a seamless experience across all mobile devices and functions efficiently as a Progressive Web App.

## Core Responsibilities

1.  **Responsive Layouts**:
    *   Design mobile-first interfaces using Tailwind CSS breakpoints.
    *   Optimize touch targets (buttons, links) for mobile users (minimum 44x44px).

2.  **PWA Integration**:
    *   Configure `manifest.json` with appropriate icons and theme colors.
    *   Implement Service Workers for offline capabilities and faster loading.
    *   Manage the "Add to Home Screen" prompt experience.

3.  **Performance Optimization**:
    *   Optimize assets (images, fonts) for fast mobile loading.
    *   Implement lazy loading for images and components.
    *   Minimize script execution to prevent mobile battery drain.

4.  **Native-like Interactions**:
    *   Implement pull-to-refresh where applicable.
    *   Use smooth, hardware-accelerated transitions.

## Key Metrics

*   Lighthouse Performance Score > 90.
*   First Contentful Paint < 1.5s on mobile.
